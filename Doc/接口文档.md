# 接口文档



#### 导出

- 接口路径 POST /v1/export 
- 请求Hearders
- 请求Body

```json
{
    "timestamp": 1624614151, // 时间戳
    "ext_type": "XLS",       // 导出文件格式[xls,xlsx]
    "title": "导出标题",       // 导出标题，查导出历史用
    "user_id": "",            // 用户标识
    "callback": "",           // 回调地址
    "source_type": "http",    // 数据源类型
    "source_http": { // http 数据源
        "url": "http:127.0.0.1:8080/demo/page",
        "method": "GET",
        "header": {},
        "param": {}
    },
    "source_sql":{ // sql数据源
        "sql": "",
        "param": []
    },
    "source_raw": { // 携带数据源
        "list": []
    }
}
```

- 返回参数

````json
````

- curl

```bash
curl --location --request POST '127.0.0.1:8080/v1/export' \
--header 'Content-Type: application/json' \
--data-raw '{"timestamp":1624614151,"ext_type":"XLS","title":"","user_id":"","call_back":"","source_type":"http","source_http":{"url":"http:127.0.0.1:8080/demo/page","method":"GET","header":{},"param":{}},"source_sql":{"sql":"","param":[]},"source_raw":{"list":[]}}'
```
